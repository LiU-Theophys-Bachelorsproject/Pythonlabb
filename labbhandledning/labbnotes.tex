\begin{shaded}
\begin{itemize}
\item[\circled{3}] Man konverterar ifrån Celsius till Fahrenheit med följande ekvation:
  $$ f = \frac{9c}{5} + 32$$
Där $f$ är temperatur i Fahrenheit, och $c$ i Celsius. Skriv en funktion som tar $c$ som parameter och returnerar $f$. 
\item[\circled{4}] Utöka funktionen i (3) till att också kunna konvertera ifrån en temperatur angiven i kelvin. Du får själv välja hur du designar gränssnittet (parametrarna till funktionen), men det måste vara en och samma funktion som kan utföra båda konverteringarna.
\item[\circled{5}] Någon vill använda Python för att numeriskt verifiera att ekvationerna
  $$ (a+b)^2 = a^2 + 2ab + b^2$$
  $$ (a-b)^2 = a^2 - 2ab + b^2$$
gäller för $a=3,3$ och $b=5,3$. De har skrivit följande program för att göra detta. Programmet innehåller dock ett antal fel. Korrigera dessa så att programmet gör vad som är tänkt.
\begin{python}
a = 3,3 b = 5,3
a2 = a**2
b2 = b**2
eq1_sum = a2 + 2ab + b2
eq2_sum = a2 - 2ab + b2
eq1_pow = (a + b)**2
eq2_pow = (a - b)**2
print('First equation: %g = %g', % (eq1_sum, eq1_pow))
print('Second equation: %h = %h', % (eq2_pow, eq2_pow))
\end{python}
\end{itemize}

 \begin{itemize}

Du ska ta ut dessa genom enbart en enda genomläsning av filen utan att läsa in hela filen i en sträng först (t.ex., genom en loop över rader ur en öppnad fil).
